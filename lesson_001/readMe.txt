
C 语言的入门，C 语言的原始文件都是以 .c 为后缀名的
  程序的入库是main 函数，函数的概念暂不做解释。
  在本例的main函数中 输出一个话 ”Hello C“ 到控制台
  演示一个简单的加法函数，实现两个数相加并返回结果，sum 函数实现了此功能。
  
lesson_001 目录下存放了一个main.c 的原始程序文件

本例的功能：计算两个数的和并输出到控制台
1 在控制台输出 Hello C
2 计算两个数的和并输出到控制台，计算 20 和  200 的和（程序中写死了）
 

执行方式
windows 环境下 执行 gcc main.c  命令生成a.exe 文件，然后执行 a.exe 

linux 或者 unix 执行 cc main.c （如果cc 命令不成功 尝试 gcc 命令）命令 生成a.out 文件，然后执行 a.out
